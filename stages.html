Stage au sein du Pôle SIG du Département du Calvados

De mai à août 2020






Le Pôle SIG (Sytèmes d'Information Géographique) du Département du Calvados a pour mission de collecter et centraliser numériquement les données géographiques départementales ainsi que les mettre à dispostion des autres services. Le Département SIG était à la recherche d'un stagiaire dans le but d'accompagner le Pôle SIG pour ses diverses tâches en cours et c'est à cette occassion que j'ai pu effectué un stage de 3 mois au sein de ce service. Le stage s'est articulé autour de 3 problèmatiques majeures.

L'appariement automatique des itinéraires de randonnées sur la BDTopo de l'IGN

Objectif : Automatiser la rectification des erreurs de tracé sur des polylignes obtenues via GPS.

Le but était de trouver un moyen de corriger des erreurs de géométrie sur des données obtenues via relevés GPS. La solution retenue a été a création d'un code SQL afin d'apparier les données GPS incorrectes sur d'autres données géométriquement correctes afin de corriger les premières.

Mise en place de géotraitements avec PGRouting

Objectif : Calculer automatiquement le chemin le plus court entre deux emplacements/Créer des isodistances autour d'un point.

PGRouting est une extension de PostGis proposant des fonctionnalités de routage géospatial et d'autres analyses de réseaux. Il permet ainsi de réaliser plusieurs géotraitements topologiques. Le Pôle SIG souhaitait réaliser des essais afin de mieux comprendre le fonctionnement des différentes fonctions de PGRouting. J'ai ainsi pu élabore un code SQL afin de calculet l'itinéraire le plus court entre deux points ou tronçons de route ainsi qu'n second code pour créer des isodistances autour d'un point.

Alimenter en données data.calvados.fr

Objectif : Diffuser des données en ligne en open data.

Le Département dispose d'une base de données SIG qui alimente ses différents outils : Mapéo Calvados, SIR, Calvados Atlas. En complément de ces outils, le Département du Calvados a souhaité mettre en ligne un catalogue de données géographiques dont certaines données seraient en libre téléchargement : data.calvados.fr.

J'ai donc utilisé le logiciel FME, un outil de traitements de données , afin d'automatiser l'extraction des données de la base de données SIG afin de permettre leur publication sur data.calvados.fr et data.gouv.fr. via des scripts.
